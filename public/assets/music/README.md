# Musiques de combat

Placez vos fichiers audio ici:

- **combat.mp3** - Musique jouée pendant le combat (en boucle)
- **victory.mp3** - Musique jouée à la fin du combat

## Formats supportés
- MP3 (recommandé)
- OGG
- WAV

## Suggestions de musiques libres de droits
- **Combat**: Musique épique/orchestrale intense
- **Victoire**: Fanfare triomphale courte (10-15 secondes)
